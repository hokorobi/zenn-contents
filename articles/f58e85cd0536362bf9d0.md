ヴィジュアルモードの判定の仕方を記録しておきたかった覚書。

```Vim
function! s:GetMode()
  if mode() ==# 'i'
    return 'i'
  endif

  if stridx('vV', mode()) > -1
    return 'v'
  endif

  return 'n'
endfunction
```

## 2022-11-25 訂正

* `mode()` は一文字しか返さないから余分な処理を削除
* 矩形ビジュアルの判定は `<C-v>` だとできていなかったので訂正
