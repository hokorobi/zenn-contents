ヴィジュアルモードの判定の仕方を記録しておきたかった覚書。

```Vim
function! s:GetMode()
  if mode() ==# 'i'
    return 'i'
  endif

  if mode() =~# '^[vV\x16]'
    return 'v'
  endif

  return 'n'
endfunction
```
