ヴィジュアルモードの判定の仕方を記録しておきたかった覚書。

```Vim
function! s:GetMode()
  if mode()[0] ==# 'i'
    return 'i'
  endif

  if stridx('vV<c-v>', mode()) > -1
    return 'v'
  endif

  return 'n'
endfunction
```
